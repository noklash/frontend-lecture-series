
# SESSION 1  
**Introduction to Frontend Development & How the Web Works**

## Learning Objectives

By the end of this session, students will be able to:

- Understand what frontend development is
- Explain how websites work
- Distinguish between frontend and backend
- Understand the role of HTML, CSS, and JavaScript
- Set up a professional development environment
- Build and run their first webpage

**Total Time:** 120 Minutes

---

## 0–20 Minutes: Introduction & Theory

### Icebreaker

**Ask students:**  
"What websites do you use every day?"

**Examples:**  
- YouTube  
- X  
- Instagram  
- Netflix  
- Jumia  
- Amazon  
- Spotify

**Follow-up question:**  
"What do all these websites have in common?"

**Answer:**  
They all have **text, buttons, images, navigation, and user interaction**.  

These visible parts are called the **Frontend**.

### What Is Frontend Development?

Frontend development is the creation of everything users can see and interact with inside a browser.

**Examples:**
- Buttons
- Forms
- Navigation menus
- Animations
- Dashboards
- Landing pages

A frontend developer transforms designs into working applications.

### Frontend vs Backend

| Aspect          | **Frontend**                          | **Backend**                              |
|-----------------|---------------------------------------|------------------------------------------|
| **What it is**  | What users see and interact with     | What users don't see                    |
| **Examples**    | Login form, Dashboard, Product page  | Databases, Authentication, APIs, Servers |
| **Technologies**| HTML, CSS, JavaScript                | Node.js, Python, Java, Go               |

### Restaurant Analogy

- **Frontend** → Dining room (where customers interact)  
- **Backend** → Kitchen (where food gets prepared)  
- **API** → Waiter (carries requests and responses between them)

### Why Frontend Matters in 2026

Modern companies compete heavily on:
- User experience
- Performance
- Accessibility
- Mobile responsiveness
- AI integration

**Leading Examples:** ChatGPT, Notion, Linear, Stripe, Airbnb.

Great frontend developers are still highly demanded because **AI can generate code, but humans design exceptional experiences**.

## 20–70 Minutes: Live Coding & Instructor Demonstration

### Step 1: Install VS Code
VS Code is the industry-standard code editor with powerful features:
- Extensions
- Git integration
- AI tools
- Debugging

### Step 2: Create Project Folder

```
frontend-bootcamp
└── index.html
```

### Step 3: Create First HTML File

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Website</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>
```

Explain every line carefully.

### Step 4: Open in Browser
- Double-click the `index.html` file, or
- Use **Live Server** extension in VS Code

### Step 5: Add More Content

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Website</title>
</head>
<body>
    <h1>Goodluck's First Website</h1>
    <p>I am learning frontend development.</p>
</body>
</html>
```

### Step 6: Browser Developer Tools
Press `F12` to open DevTools and explore:
- **Elements** panel
- **Console**
- **Network** tab

> Developer Tools are a frontend developer's best friend.

## 100–110 Minutes: Review, Common Mistakes & Q&A

### Review Questions

- What does **HTML** stand for?  
  → **HyperText Markup Language**

- What does **frontend** mean?  
  → Everything users see and interact with.

- What language controls styling?  
  → **CSS**

- What language controls behavior?  
  → **JavaScript**

### Common Mistakes

- **Missing closing tags**  
  Wrong: `<p>Hello`  
  Correct: `<p>Hello</p>`

- **Forgetting to save** → Always use `CTRL + S`

- **Incorrect file extension**  
  Wrong: `index.txt`  
  Correct: `index.html`

## Key Concepts & Teaching Notes

- **HTML** → Structure (the **skeleton** of a website)
- **CSS** → Appearance (the **clothing and makeup**)
- **JavaScript** → Behavior (the **muscles and movement**)

The **Browser** interprets HTML, CSS, and JavaScript to render the final webpage.

### Live Coding Example (Final Version)

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Website</title>
</head>
<body>
    <h1>Welcome To Frontend Development</h1>
    <p>This is my first webpage.</p>
</body>
</html>
```
